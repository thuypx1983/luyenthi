 * Current maintenance by multiple MoneySuite users.
 * Re: new initiative: https://www.drupal.org/node/2315653 

Requirements:
-------------
-Drupal 7
-Views (http://drupal.org/project/views)
-Token (http://drupal.org/project/token)
-MS Core (https://www.moneyscripts.net/ms_core)
-Rules (http://drupal.org/project/rules)
-Date (http://drupal.org/project/date)
-Entity API (http://drupal.org/project/entity)

Description:
------------
Events Ticketing Suite lets you create event fields and attach them to any entities,
such as nodes or users. Customers can then purchase tickets to the event. The
Seatmaps submodule lets you quickly and easily create seating charts that users
can use when selecting which seats they want.

Installation:
-------------
Install the module just like you would any other Drupal Module
